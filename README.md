# AreYouPalm
## 야 자냐? - 문자 보내주는 알람앱
An alarm app that sends a text message to a designated person.

기본 기능
- 알람
- 요일/일자 선택 및 반복 기능
- 진동/알람 음량 선택
- 스와프해서 알람 삭제

주요 기능
- 알람 끄기 실패시 문자를 보낼 상대 및 메시지 지정 가능
